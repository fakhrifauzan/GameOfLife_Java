# Game Of Life

## Description

This program can simulating the Conway's Game of Life.

## Environment Setup
- Java 11 (OpenJDK)
- Gradle 5.2.1
- JUnit 4

## Build Instruction

To build this project, run this command
```bash
./gradlew build
```

## Run Test
To run test, please run this command
```bash
./gradlew check
```
